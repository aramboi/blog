Title: Getting the latest Firefox release for 64bit Linux
Date: 2011-06-09 12:24
Category: misc
Tags: firefox beta, linux
Summary: With working Flash Player? Please, tell me more.

On the usual channels (like
[http://www.mozilla.com/en-US/firefox/channel/][] ) you can only
download the 32bit version of the latest Firefox release.

But I stumbled over a public Mozilla FTP where they hold all releases,
including the 64bit one.

Link: [ftp://ftp.mozilla.org/pub/mozilla.org/firefox/releases/][]

This way all the installed plug-ins on your 64bit \*nix system will work
out of the box (including Adobe Flash).
